# EEIMS — Electric Equipment Inventory Management System

A full stack web application to manage electrical equipment in a manufacturing factory.

## Tech Stack
- Frontend: React.js (Vite)
- Backend: Node.js + Express.js
- Database: MySQL + Sequelize

## Roles
- Super Admin
- Manager
- Technician

## Setup
### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
